More Tidyverse

library(tidyverse)
library(lubridate)
set.seed(1234)

The tidyverse is a collection of packages built for a common style of data analysis. The idea is simple: keep data in tidy columns, then solve tasks by chaining clear verbs.

Some data to work on

n <- 1000
df <- tibble(
  id = 1:n,
  date = sample(seq(ymd("2024-01-01"), ymd("2024-12-31"), by = "days"), n, replace = TRUE),
  age = rnorm(n, mean = 40, sd = 10) |> round(),
  sex = sample(c("M", "F"), n, replace = TRUE),
  weight = rnorm(n, mean = 75, sd = 10) |> round(),
  height = (rnorm(n, mean = 170, sd = 7) + (date - ymd("2024-01-01")) / 36.5 - (sex == "F")*7) |> as.numeric() |>round(),
  note = sample(c("new-patient", "follow-up", "urgent"), n, replace = TRUE)
)

dplyr in one minute

{dplyr} is for data manipulation with readable verbs. Most common pattern: data |> filter(...) |> mutate(...) |> summarise(...).

tidyr: reshape data

{tidyr} helps move between wide and long data and fill missing values.

Other tidyverse packages (quick tour)

readr (import text data)

Purpose: fast, friendly file import. Useful functions: read_csv(), read_tsv(), write_csv().

Tip

Other nice packages are {readxl} for Excel files and {googlesheets4} for Google Sheets along with {tidyxl} for non-tabular Excel files.

purrr (iterate without explicit loops)

Purpose: apply functions consistently. Useful functions: map(), map_dbl(), map2().

df |>
  select(weight, height) |>
  summarise(across(everything(), list(mean = mean, sd = sd)))
# A tibble: 1 × 4

ggplot2 teaser (full lesson later)

One line summary: {ggplot2} builds plots by adding layers.

df |>
  ggplot(aes(x = date, y = height, color = sex)) +
  geom_point(size = 3, alpha = 0.9) +
  geom_smooth(method = "loess", se = FALSE, linewidth = 1) +
  scale_color_brewer(palette = "Set1") +
  labs(
    title = "Height over time",
    subtitle = "Lesson 7 teaser: grammar of graphics",
    x = NULL,
    y = "Height (cm)",
    color = NULL
  ) +
  theme_minimal(base_size = 13) +
  theme(legend.position = "top")
`geom_smooth()` using formula = 'y ~ x'
